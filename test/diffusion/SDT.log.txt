it should be noted that the calculation order will slightly affect the sum of the array of type float(double):
see the following example:
----
    static inline void Compute_rho(const T *f, T &rho) {
    rho = T(0);
    for (int i = 0; i < LatSet::q; i++) {
      rho += f[i];
    }
  }
----
